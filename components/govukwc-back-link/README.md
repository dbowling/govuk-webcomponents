# govukwc-back-link

## Usage

```javascript
import 'govukwc-webcomponents/components/govukwc-back-link/govukwc-back-link';
```

## API

## `<govukwc-back-link>`

### Properties

| Property  |  Type     | Default | Description |
|-----------|-----------|---------|-------------|
| url|string|""|The back link url
| label|string|"Back"|The back link label|
